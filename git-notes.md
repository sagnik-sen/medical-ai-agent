# Git Notes

## Commands
- `git branch`: Shows branches
- `git checkout -b <branch>`: Creates & switches
- `git merge <branch>`: Merges into current branch
- `git push`: Pushes local commits to GitHub
- `git pull`: Pulls latest changes from GitHub
